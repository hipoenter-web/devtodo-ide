# Sample Projects

과제 시연용 샘플 프로젝트를 넣는 폴더입니다.

## 사용 방법

이 폴더 안에 광고주 또는 프로젝트별 폴더를 추가합니다.

```text
sample-projects/
├── hypo-interactive-site/
│   ├── index.html
│   ├── style.css
│   └── images/
├── hospital-landing/
│   └── index.html
└── detail-page-review/
    ├── 01-main.png
    ├── 02-detail.png
    └── 03-footer.png
```

## Preview 기준

- `index.html`이 있는 폴더는 정적 HTML Preview 대상으로 사용합니다.
- 이미지/GIF/영상 파일이 있는 폴더는 파일명 순서대로 긴 상세페이지처럼 Preview합니다.
- 순서가 맞지 않는 경우 앱 Preview 안의 위/아래 버튼으로 순서를 조정할 수 있습니다.
- React/Vite 프로젝트는 서버 실행 또는 배포 URL 연결 대상으로 분류합니다.

향후 백엔드가 연결되면 이 폴더 구조를 서버의 프로젝트 목록/권한 데이터로 대체할 수 있습니다.
